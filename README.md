## Description
A modern, interactive password generator built with HTML, CSS, and JavaScript. It allows users to generate strong and secure passwords with customizable options. The project features a sleek interface with dark mode, password strength indicator, saved passwords, and easy copy functionality.

---

## Features
- **Custom Password Length:** Choose a length between 4 and 64 characters.  
- **Character Options:** Include uppercase, lowercase, numbers, and symbols.  
- **Show/Hide Password:** Toggle visibility of the generated password.  
- **Copy to Clipboard:** Quickly copy the password with a single click.  
- **Regenerate Button:** Generate a new password with the same selected options.  
- **Password Strength Indicator:** Visual bar showing the strength of the password.  
- **Saved Passwords:** Automatically save the last 3 generated passwords for quick access.  
- **Clear Saved Passwords:** Remove all saved passwords with a button click.  
- **Dark/Light Mode Toggle:** Switch between light and dark themes.  
- **Responsive & Modern UI:** Designed for clarity, ease of use, and mobile-friendly layout.

---

## How to Use
1. Open `index.html` in any modern web browser.  
2. Select your desired password options (uppercase, lowercase, numbers, symbols).  
3. Set the desired password length.  
4. Click **Generate Password** to create a new password.  
5. Use the **eye icon** to show or hide the password.  
6. Click **Copy** to copy the password to the clipboard.  
7. Use the **regenerate button** üîÑ to create a new password with the same settings.  
8. Saved passwords will appear below; you can copy or clear them.  
9. Toggle dark/light mode using the moon button at the top-right corner.
